弹窗滚动条在ipad上面的兼容性解决方案

CSS 样式

滚动条的css样式存在兼容性问题，可以设置它的样式：
::-webkit-scrollbar/整体部分/
{
width: 3px;
height:3px;
}

::-webkit-scrollbar-track/*滑动轨道*/
{
-webkit-box-shadow: inset 0 0 3px rgba(0,0,0,0.2);
border-radius: 0px;
background: rgba(0,0,0,0.1);
}

::-webkit-scrollbar-thumb/*滑块*/
{
border-radius: 3px;
-webkit-box-shadow: inset 0 0 3px rgba(0,0,0,0.2);
background: rgba(0,0,0,0.2);
}

::-webkit-scrollbar-thumb:hover/*滑块效果*/
{
border-radius: 3px;
-webkit-box-shadow: inset 0 0 3px rgba(0,0,0,0.2);
background: rgba(0,0,0,0.4);
}

如果需要实现隐藏横向滚动条等具体需求需要在使用的地方设置，overflow-x:hidden
（hidden,裁剪内容，不提供滚动条/auto,如果溢出，提供滚动机制/Visible:不裁剪内容，显示在内容框之外）

touch.js

弹窗在ipad上面的滚动效果存在兼容性问题，滚动条滚动到弹窗的顶部和底部的时候弹窗后面的屏幕会滚动（蒙层区域）
需要禁止弹框中的滚动条，滚动到顶部以上和底部以下区域的touch事件，具体的实现方法是拿到滚动部分的clientHeight（弹框的高度）,和scrollHeight（可滚动区域的总高）,计算出maxHeight（可滚动的高度）.

touch.js
const touch = (el, fn) => {
let startX, startY;
el.addEventListener(‘touchstart’, e => {
const start = e.targetTouches[0];
startX = start.pageX;
startY = start.pageY;
});

el.addEventListener('touchmove', e => {
const end = e.changedTouches[0];
const moveX = end.pageX - startX;
const moveY = end.pageY - startY;
fn(moveX, moveY, e);
});

}

let scrollpart = $(‘.agreement-content’)[0];
touch(scrollpart, (x, y, e) => {
const curScrollHeight = scrollpart.scrollTop;
const maxScrollHeight = scrollpart.scrollHeight - scrollpart.clientHeight;
if (!maxScrollHeight) {
e.preventDefault();
} else if (curScrollHeight <=0 && y > 0) {
e.preventDefault();
} else if (curScrollHeight >= maxScrollHeight && y < 0) {
e.preventDefault();
}
});
($(‘.overlay’)[0]).addEventListener(‘touchStart’, e => {
e.preventDefault();
});
($(‘.overlay’)[0]).addEventListener(‘touchmove’, e => {
e.preventDefault();
});
},

蒙层区域的兼容性处理

一般蒙层区域的样式处理是在弹框出来的时候显示蒙层，蒙层的样式：
{
position: fixed;
z-index: 99990;
top: 0;
right: 0;
bottom: 0;
left: 0;
background-color: #000;
opacity: .4;
}
同时给body元素的样式上添加样式：
{
height:100%;overflow:hidden
}
防止页面的滚动，但是在触屏端需要做兼容性处理: 取蒙层区域(overlay)，在touchStart和touchmove事件中禁用touch

($('.overlay')[0]).addEventListener('touchStart', e => {
     e.preventDefault();
});

 ($('.overlay')[0]).addEventListener('touchmove', e => {
     e.preventDefault();
 });
